# OpenRouterChats

## English Version

### About the Application
OpenRouterChats is an AI-powered educational assistant that helps students prepare for exams and complete homework assignments. The application features:

- **Customizable AI Teachers**: Store and manage prompts for different teacher personas
- **Multiple LLM Models**: Choose from various language models to process your queries
- **Personalized Learning**: Add your own teachers and customize their behavior

The app acts as a bridge between students and AI tutors, providing structured interactions with different teaching styles and approaches.

### Key Files
- **update_app.sh**: Script for updating the application components
- **UPDATE_GUIDE.md**: Instructions for updating the application manually
- **DEPLOYMENT_GUIDE.md**: Step-by-step guide for deploying the application

### Technology Stack
- Backend: Django (Python)
- Database: SQLite
- Deployment: WSGI (Gunicorn)
- Frontend: Django Templates
- AI Integration: OpenRouter API

## Русская Версия

### О Приложении
OpenRouterChats - это образовательный помощник на основе ИИ, который помогает студентам готовиться к экзаменам и выполнять домашние задания. Основные возможности:

- **Настраиваемые AI-учителя**: Хранение и управление промптами для разных преподавателей
- **Множество моделей ИИ**: Выбор из различных языковых моделей для обработки запросов
- **Персонализированное обучение**: Добавляйте своих учителей и настраивайте их поведение

Приложение выступает в роли посредника между студентами и ИИ-преподавателями, обеспечивая структурированное взаимодействие с разными стилями преподавания.

### Ключевые Файлы
- **update_app.sh**: Скрипт для обновления компонентов приложения
- **UPDATE_GUIDE.md**: Инструкции по ручному обновлению приложения
- **DEPLOYMENT_GUIDE.md**: Пошаговое руководство по развертыванию приложения

### Технологический Стек
- Бэкенд: Django (Python)
- База данных: SQLite
- Развертывание: WSGI (Gunicorn)
- Фронтенд: Django Templates
- Интеграция с ИИ: OpenRouter API
